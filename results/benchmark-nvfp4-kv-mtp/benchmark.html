Six Root-Caused Fixes
1OOM Killer → MAX_JOBS=6
20 parallel nvcc exhausted 121 GB RAM. Reduced to 6 + NVCC_THREADS=2. Peak ~15 GB.
2Missing Packages → Bulk site-packages COPY
Cascading ModuleNotFoundError. Single bulk COPY from builder solved all deps.
3PTX Mismatch → CUDA 13.0 (not 13.2)
nvcc 13.2 PTX incompatible with PyTorch CUDA 13.0 runtime. Full rebuild.
4No C Compiler → build-essential
FlashInfer JIT needs gcc at runtime.
5No Ninja → ninja-build
FlashInfer JIT uses ninja as build system.
6Missing CUDA Dev Headers → cuda-libraries-dev-13-0
JIT kernels need curand_kernel.h at runtime.
